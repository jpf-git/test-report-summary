===============================================
  GitHub Pages 上传操作步骤（超简单版）
===============================================

当前文件夹里的所有内容都需要上传到GitHub！

===== 第1步：上传文件到GitHub =====

1. 打开你的浏览器
2. 访问：https://github.com/jpf-git/test-report-summary
3. 在页面上找到"Add file"按钮，点击下拉菜单选择"Upload files"
4. 把当前文件夹里的所有内容拖拽到网页上
   （包括：index.html, README.md, 4个报告文件夹）
5. 下方写提交信息：Add performance test reports
6. 点击绿色按钮"Commit changes"
7. 等待上传完成（可能需要几分钟）

===== 第2步：启用GitHub Pages =====

1. 在仓库页面，点击顶部的"Settings"（设置）
2. 左侧菜单找到"Pages"
3. 在"Source"下：
   - Branch: 选择 main（或master）
   - 文件夹: 选择 / (root)
   - 点击Save按钮
4. 等待1-3分钟，页面会显示你的网址

===== 第3步：访问你的在线报告 =====

你的报告网址：
https://jpf-git.github.io/test-report-summary/

把这个链接发给任何人，他们都能在浏览器里看到你的测试报告！

===== 注意事项 =====

1. 上传时确保4个report文件夹里的所有内容都完整上传
2. 启用Pages后等待3-5分钟再访问链接
3. 如果显示404，清除浏览器缓存再试
